## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, gap-1 |
| sm | 8px | Compact element spacing, gap-2, p-2 |
| md | 16px | Default element spacing, p-4 |
| lg | 24px | Section padding, p-6 |
| xl | 32px | Layout gaps |
| 2xl | 48px | HOUR_HEIGHT in calendar grid (existing constant) |
| 3xl | 64px | Not used in this phase |

Exceptions:
- Lane header height: 20px (5 * 4px) -- fits a single line of text-[10px] with padding
- Today indicator dot: 8px (2 * 4px) -- visible but unobtrusive
- Conflict badge: 16px min-width (4 * 4px) -- fits single-digit count

## Interaction Contracts

### Quick-Add Button (FLOW-01, D-01 through D-05)

| State | Visual | Behavior |
|-------|--------|----------|
| Default | `Plus` icon, 14px, `text-muted-foreground` | Visible at right edge of every tutor row |
| Hover | `text-primary` | Cursor pointer |
| Click | Flashes to `Check` icon in `text-available` for 800ms, then reverts to `Plus` | Calls `addTutor(id, name)` immediately |
| Disabled (3 tutors selected) | `Plus` icon, `opacity-40`, `cursor-not-allowed` | Native `title="Remove a tutor first (max 3)"` |
| Already added | `Check` icon, `text-available`, static | No click handler |

Button spec:
- Size: `h-6 w-6` (24px touch target)
- Border: none
- Background: transparent, `hover:bg-muted` rounded
- Position: rightmost column in availability grid table, `text-center`
- `e.stopPropagation()` on click to prevent row selection

### Fullscreen Toggle (D-06 through D-09)

| State | Visual | Behavior |
|-------|--------|----------|
| Split view (default) | `Maximize2` icon, 14px, `text-muted-foreground` | Click enters fullscreen |
| Fullscreen | `Minimize2` icon, 14px, `text-muted-foreground` | Click exits fullscreen |
| Hover (both) | `text-foreground` | Cursor pointer |
| Esc key (fullscreen) | n/a | Exits fullscreen |

Button spec:
- Size: `h-7 w-7` (28px)
- Variant: ghost (transparent bg, `hover:bg-muted`)
- Position: in compare panel header area, right-aligned alongside "N/3" counter
- `aria-label`: "Enter fullscreen" / "Exit fullscreen" (toggles with state)

Layout transition:
- Search panel: `transition-all duration-300 ease-in-out`
- Split view: `w-1/2` with `border-r border-border/50 pr-3`
- Fullscreen: `w-0 opacity-0 pr-0 border-r-0 overflow-hidden`
- Compare panel: `transition-all duration-300 ease-in-out`
- Split view: `w-1/2 pl-1`
- Fullscreen: `w-full pl-0`
- Search panel remains mounted (DOM preserved, React state intact)

### Lane Identity (CAL-01, CAL-02, D-10, D-11)

**Lane tint background (CAL-01):**
- Full-height absolute-positioned div behind session cards
- `pointer-events-none` to not interfere with card clicks
- Background: `rgba(TUTOR_COLORS[idx], 0.05)` via existing `rgba()` helper
- Width: proportional to lane width (`laneWidthPct`)
- z-index: `z-0` (below session cards at `z-[1]`, below popovers at `z-50`)
- Only rendered in multi-tutor view (2-3 tutors)

**Lane header label (CAL-02):**
- `position: sticky` with `top: 0` inside the `overflow-y-auto` scroll container
- z-index: `z-[5]` (above session cards, below popovers)
- Height: 20px
- Background: `bg-background/90 backdrop-blur-sm` for readability over scrolling content
- Content: color dot (6px circle, `rounded-full`) + tutor display name (truncated)
- Font: `text-[10px] font-medium` in tutor's assigned color
- Only rendered in multi-tutor view (2-3 tutors)

### Today Indicator (CAL-03, D-13, D-14)

**Visibility rule:** Only rendered when `weekStart === getCurrentMonday()` (current week).

**Line spec:**
- Horizontal line spanning full width of today's day column
- Height: 2px
- Color: `bg-red-500`
- z-index: `z-[3]` (above lane tints, below session cards)
- Position: absolute, calculated via `minuteToY(currentMinuteInBangkok)`

**Dot spec:**
- Circle at left edge of the line
- Size: 8px (`h-2 w-2`)
- Color: `bg-red-500`
- Shape: `rounded-full`
- Position: absolute, centered vertically on the line, left edge of column

**Timer:**
- `useState` for current minute + day-of-week
- `useEffect` with `setInterval` at 60000ms (1-minute interval)
- Timezone: `new Date(new Date().toLocaleString("en-US", { timeZone: "Asia/Bangkok" }))` -- matches existing `getCurrentMonday()` pattern
- Cleanup: `return () => clearInterval(id)` in useEffect

### Conflict Count Badge (CAL-04, D-15, D-16)

**Day tab badge (ComparePanel):**
- Inline span after day tab text, e.g. "Mon 13/4" followed by badge
- Shape: `rounded-full` pill
- Size: `h-4 min-w-[16px]` (16px height, 16px minimum width)
- Padding: `px-1` (4px horizontal)
- Background: `bg-conflict`
- Text: `text-white text-[10px] font-semibold`
- Content: conflict count number (integer)
- Margin: `ml-1` from day text
- Only shown when count > 0

**Week overview header badge (WeekOverview):**
- Replaces existing `<span className="ml-1 text-conflict">!</span>` at line 261
- Same visual spec as day tab badge above
- Replaces generic "!" with actual count number

Conflict count computation:
- ComparePanel: `useMemo` over `compareResponse.conflicts`, group by `dayOfWeek`, count per day
- WeekOverview: reuse existing `conflictsByDay` map, read `.length` for count

### Session Hover Tooltip (FLOW-02)

- Implementation: native HTML `title` attribute on session card `<button>` elements
- Content format: `"{studentName} - {subject}\n{startTime}-{endTime}"`
- No new component dependency (no shadcn Tooltip needed)
- Applied to both WeekOverview and CalendarGrid session cards

### Week URL Parameter (FLOW-03)

- Parameter: `?week=YYYY-MM-DD` (ISO date format, Monday of the target week)
- Sync method: `window.history.replaceState` (non-navigating, no re-render)
- On mount: read `?week=` from `useSearchParams`, pass to `changeWeek` alongside existing `?tutors=` handling
- On change: update URL whenever `weekStart` or `compareTutors` change
- Default week: parameter omitted when viewing current week (clean URL)

### Keyboard Navigation (FLOW-04)

- Scope: `window` keydown listener
- `ArrowLeft`: navigate to previous week (calls `changeWeek(shiftWeek(weekStart, -1))`)
- `ArrowRight`: navigate to next week (calls `changeWeek(shiftWeek(weekStart, 1))`)
- Guard: skip when `e.target instanceof HTMLInputElement || e.target instanceof HTMLTextAreaElement`
- `e.preventDefault()` to suppress browser default scroll behavior
