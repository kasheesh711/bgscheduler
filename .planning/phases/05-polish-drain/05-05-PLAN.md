From src/lib/search/recommend.ts:

```typescript
export interface RecommendedSlot {
  id: string;
  subSlotIndex: number;
  start: string;
  end: string;
  availableTutors: RangeGridRow[];
  confidence: "Best fit" | "Strong fit" | "Good fit";
  reasons: string[];
}

export function getRecommendedSlots(
  response: RangeSearchResponse,
  limit = 3,
): RecommendedSlot[];
```

The function:
1. Returns [] when response is null/undefined, has zero subSlots, or zero grid rows.
2. Maps each subSlot to `{ subSlotIndex, start, end, availableTutors }` where availableTutors = rows where row.availability[i] === true.
3. Filters out entries with zero available tutors.
4. Sorts by availableTutors.length DESC, tie-break by start time ASC (localeCompare).
5. Slices to `limit` (default 3).
6. Assigns confidence: index 0 → "Best fit", index 1 → "Strong fit", index ≥ 2 → "Good fit".
7. Builds reasons[]: always includes `"N qualified tutor(s) free"`; adds modality reason based on union of `tutors[].supportedModes`:
   - both online + onsite → "Online + onsite options"
   - online only → "Online only"
   - onsite only → "Onsite only"
   - neither (edge: modes set is empty) → no modality reason added
   Also adds "Variety to offer parent" when tutors.length >= 3.
