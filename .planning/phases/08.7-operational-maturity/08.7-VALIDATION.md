## Sampling Rate

- **After every task commit:** run the narrowest relevant unit test, or `npm test` if the task touches shared route/sync behavior.
- **After every plan wave:** run `npm test`; run `npm run test:integration` after sync/pruning changes.
- **Before `/gsd-verify-work`:** `npm test`, `npm run test:integration`, and `npx tsc --noEmit --project tsconfig.json` must pass.
- **Max feedback latency:** keep normal task feedback under 60 seconds; integration feedback under 90 seconds.
