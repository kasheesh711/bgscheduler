## Layout Contract (Critical -- Must Be Preserved Exactly)

The following layout structure is the exact class composition from the current monolith. When extracting into SearchWorkspace, SearchForm, SearchResults, and ComparePanel, these classes MUST NOT change.

### Top-level split (SearchWorkspace)
```
div.flex-1.flex.gap-3.overflow-hidden.min-h-0
  div.w-1/2.flex.flex-col.overflow-hidden.min-w-0.border-r.border-border/50.pr-3
    [SearchForm]
    [Error banner: div.rounded-md.bg-destructive/10.p-2.text-xs.text-destructive.mt-2]
    [SearchResults]
  div.w-1/2.flex.flex-col.overflow-hidden.min-w-0.pl-1
    [ComparePanel]
```

### Key constraints
- Both panels are `w-1/2` with `overflow-hidden` and `min-w-0`
- Left panel has `border-r border-border/50 pr-3`
- Right panel has `pl-1`
- Gap between panels is `gap-3` (12px)
- Parent is `flex-1` to fill remaining viewport after AppNav
- `min-h-0` prevents flex children from overflowing

### Suspense wrapper (page.tsx default export)
```
Suspense fallback={div.py-8.text-center.text-muted-foreground "Loading..."}
  SearchWorkspace (replaces SearchPageInner)
```

6. **Copy preserved** -- all user-visible strings are identical
